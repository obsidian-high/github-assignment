## Key git commands i used

 Command & Description 
 `git clone <repo-url>`  Downloads a GitHub repo to my computer 
 `git add .`  Stages all changes for commit 
 `git commit -m "message"`  Saves a version snapshot with a message 
 `git push origin main`  Uploads my committed changes back to GitHub 
 `git status`  Shows tracked changes 
 `git pull`  Fetches latest updates from remote repo
